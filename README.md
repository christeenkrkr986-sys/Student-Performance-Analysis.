# Student Performance Analysis Dashboard

## Project Overview
This project involves a detailed analysis of student academic records. The goal was to monitor performance metrics, calculate grade distributions, and evaluate department-wise success rates.

## Key Features
- **GPA Calculation:** Automated grading system based on total scores.
- **Department Analysis:** Comparison between CS, IS, and IT departments.
- **Interactive Dashboard:** Built using Pivot Tables and Slicers for dynamic filtering.
